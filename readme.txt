Starward Rank Tracker - (ver 0.0.1)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

この度は「Starward Rank Tracker」をご利用いただきありがとうございます。
本ツールは、「星の翼（Starward）」のランクマッチ等の対戦ログ（RP変動・詳細スタッツ）を
自動で記録してJSONL形式で保存し、リアルタイムにセッション内の推移をグラフ化する戦績管理補助ツールです。

--------------------------------------------------------------------------------
【1. 動作環境・前提条件】
--------------------------------------------------------------------------------
本ツールは、画面上の特定の座標を画像認識して動作します。
以下の環境以外では正しく動作しない可能性があります。

・OS: Windows 10 / 11 (64bit)
・解像度: 1920x1080 (Full HD) 
・アスペクト比: 16:9 専用
　※16:10（Steam Deck等）やウルトラワイドモニターには現在対応しておりません。
・Windowsディスプレイ設定: 「拡大縮小とレイアウト」を 100% に設定してください。
・ゲーム内設定: 
　- ボーダーレスウィンドウまたはウィンドウモード
　- ステージやUIの重なりによって認識精度が落ちる場合があります。

--------------------------------------------------------------------------------
【2. 導入方法】
--------------------------------------------------------------------------------
1. ダウンロードしたZIPファイルを任意のフォルダに展開します。
2. 「starwardranktracker.exe」を実行します。
3. ツールが起動し、画面が待機状態になったら準備完了です。
4. ゲームを起動してください。

--------------------------------------------------------------------------------
【3. 機能とログの見方】
--------------------------------------------------------------------------------
■ リアルタイム・セッション推移グラフ & 勝敗表
ツールを起動して、最初に検出した「1試合目のマッチ画面」の数値を自動的に「0戦目（本日のスタート地点）」として確定します。
過去のログファイルを読み込む必要がないメモリ完結型ロジックのため、ファイル破損等によるクラッシュが起きず、軽量かつ頑丈に動作します。

・勝敗表UI: リザルト画面で保存されるとカウント（○勝○敗・勝率）が更新されます。
・推移グラフ: 「次の試合のマッチ画面」に進んで正確なRPが確定した瞬間にプロットされます。

 自動記録（JSONL形式）
　ランクマッチ等のリザルト画面を監視し、アプリと同じフォルダ内に月別のログファイル（例: `log_202605.jsonl`）を自動作成・追記します。

■ ログ（JSONL）の項目説明
1行が1試合のデータ（JSON形式）として保存されます。
　- id           : 試合の固有ID
　- datetime     : 記録日時 (YYYY-MM-DD HH:MM:SS)
　- result       : 勝敗結果 (win / lose)
　- rank_point   : 試合前のランクポイント (RP)
　- player_team  : 自チームの情報
　　└ player     : あなたのスタッツ（chara / damage / kill / death / lock）
　　└ player2    : 味方のスタッツ
　- enemy_team   : 敵チームの情報
　　└ enemy      : 敵1のスタッツ
　　└ enemy2     : 敵2のスタッツ

※「lock」は対戦中のターゲットロック関連のスタッツ項目です。

■ページ切り替え
画面左上のボタンでグラフ表示とログ分析画面を切り替えます

■分析画面の説明
1. マイキャラ軸スタッツ
あなたが使用したキャラクターごとの総合戦績と、相方（味方）との相性を分析する画面です。
使用キャラカード（試合数順に自動ソート）: あなたが使ったキャラごとに、勝率、勝敗数、総試合数が並びます。
勝率が50%以上の得意キャラはブルー、50%未満はオレンジでハイライトされます。
個人スタッツ概要: カードの表面には、そのキャラを操作した際の「平均ダメージ」「チーム内ダメージシェア（%）」
「平均キル／デス（K/D）」「平均ターゲットロック率」が常時表示されます。 
 ▼ 相方別の詳細相性スタッツ（開閉式）: 
 キャラカードをクリックすると、「どの相方キャラと組んだ時に勝率が良いか」をまとめた詳細なデータテーブルが展開されます。
 相方ごとの平均DMGやダメシェア、平均K/D、平均ロック率を網羅しています。
2. 対戦相手軸分析
敵キャラ遭遇率（採用率順に自動ソート）: 敵チーム内でのキャラクター遭遇回数と採用率（%）、およびそのキャラが敵にいた時のあなたの勝率を表示します。
▼ 詳細相性スタッツ（開閉式）: 敵キャラカードを展開すると、「そのキャラが相手にいる時、
自分と相方が何のキャラを出していたら勝てているか」という、自分×相方の組み合わせ別の対面勝率や詳細スタッツが一覧で確認できます。
3. コスト編成分析   『星の翼』の肝である、機体コスト（3.0 / 2.5 / 2.0 / 1.5）の組み合わせシナジーを分析する独自の画面です。
自チームのコスト編成別勝率 (運用シナジー): 「3.0 + 2.0」や「2.5 + 2.5」など、自チームのコスト組み合わせごとの試合数・勝敗・勝率・平均DMGを算出。
さらに「平均被撃墜数（合計）」として、自分と相方がそれぞれ平均何回落とされたかも可視化し、前衛・後衛のコスト配分が機能しているかを分析できます。
敵チームのコスト編成別相性 (対面勝率): 相手チームがどのようなコスト編成で来ることが多いか（遭遇数）、
またそれらの編成に対するあなたの勝率や、敵チームの平均総ダメージをまとめて対策を練ることができます 。
 4. 試合履歴   直近の対戦ログを、最新の試合から順に時系列（降順）で確認できるタイムライン型ログです。
▼ 試合詳細フルスタッツ（開閉式）: 各試合のカードをクリックすると、その試合に参加したプレイヤー4人全員のスタッツを表示します。

■プロファイル管理機能について

本ツールでは、複数の「プロファイル」を作成して戦績データを完全に切り分けて管理することができます。
「シャッフル用」「特定のチーム戦用」など、目的に応じて自由な単位でログを記録・分析することが可能です。

▼使い方

1. プロファイルの切り替え
メイン画面上部の**「プロファイル」ドロップダウン**から、切り替えたいプロファイル名を選択してください。
該当するプロファイルの戦績データ（グラフや対戦履歴）へ表示が切り替わります。

2. プロファイルの追加・編集（管理ダイアログ）
ドロップダウンの右側にある**「編集（鉛筆アイコン）」ボタン**をクリックすると、管理ダイアログが表示されます。

* **プロファイルの追加**
  「追加するプロファイル名」を入力し、【追加】ボタンを押すと新しいログフォルダが作成されます。
* **プロファイルの削除**
  削除したいプロファイルをドロップダウンから選択し、【削除】ボタンを押します。

▼ 注意点
* プロファイル名にWindowsのフォルダ名として使えない特殊文字（`\` `/` `:` `*` `?` `"` `<` `>` `|` など）を使用すると、
正常にフォルダが作成されない場合があります。
* プロファイルの削除時、対象のログファイルを別のテキストエディタやExcel等で開いていると、
システムロックがかかり削除（リネーム）に失敗することがあります。その場合は、ファイルを閉じてから再度お試しください。

【4. 注意事項・免責事項】
--------------------------------------------------------------------------------
■ 画像認識する範囲について
・対戦待ち・マッチ画面のRP表示部分
・リザルト画面の勝敗文字および各種スタッツ・RP変動部分
上記部分の画面が他のウィンドウ等で隠れている状況では正常に動作しません。

■ 認識精度について
・本ツールは画面のキャプチャ情報を画像解析して数値を読み取ります。そのため、100%の記録精度を保証するものではありません。
・以下のような条件下では、読み取りミスや記録漏れが発生しやすくなります。
　- ステージ背景が非常に明るい、またはUIと文字色が重なっている場合
　- 画面に通知ポップアップ（Windows通知等）が重なった場合
　- PCの負荷が高く、キャプチャのタイミングがズレた場合

■ 免責事項
・本ツールは個人の戦績管理を補助するためのものであり、データの正確性・完全性について制作者は一切の責任を負いません。
・本ツールの利用により生じた損害、トラブルについて制作者は一切の責任を負いかねます。自己責任でご利用ください。
・本ツールは画面情報を読み取る「外部ツール」であり、ゲームメモリの改ざんや不正な通信は一切行いません。

※ゲームプレイ中はWindowsの仕様によりツールの画面更新が一時的に保留される場合があります。
ツールをクリックしてアクティブにすると最新の状態に更新されますが、裏での戦績記録は正常に行われていますのでご安心ください。
※初回起動時はファイルを展開するまで時間がかかります。動きがなくても少々お待ちください。

【5. 困ったときは（FAQ）】
--------------------------------------------------------------------------------
Q. 数値が正しく記録されない、または変な数字がログに残る
A. 本ツールは画面の「見たまま」を読み取る性質上、誤認識を完全にゼロにすることはできません。
   誤ったログが残った場合は、お手数ですが直接 `.jsonl` ファイルをテキストエディタ（メモ帳やVSCode等）で開き、該当する行を手動で修正・削除してください。
   なお、内部のグラフシステムはファイルではなく画面の数値を直接見ているため、ファイルが一時的に汚れてもグラフの動作に悪影響はありません。
